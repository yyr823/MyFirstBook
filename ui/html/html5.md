> #### HTML5文档有如下结构

```HTML
<!DOCTYPE html>
<html>
	<head>
	<meta charset="UTF-8">
	<title>页面标题</title>
	</head>
	<body>
	页面内容
	</body>
</html>
```

* **注意:**

* [x] **不要在`<html>`和`<head> `之间插入任何内容**
* [x] **不要在`<head>`和`<body>` 之间插入任何内容**
* [x] **不要在`<body>`和`<html>` 之间插入任何内容**
